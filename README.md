#### Gin-Quasar-Admin 使用 Go+Gin+Vue3+Quasar2 搭建，可以通过简单的配置快速形成自己的网站，并通过插件形式开发个性化逻辑。

### 主要功能：

1. 动态配置网站浏览器标签页图标。
2. 动态配置网站 主标题、次标题、网站描述等。
3. 动态配置后台基础设置，如验证码、JWT、上传图片/文件允许大小、允许后缀。
4. 用户可以同时拥有多个角色。
5. 角色授权菜单和api接口。
6. 部门管理、字典管理、用户管理、菜单管理等。
7. i18n。
